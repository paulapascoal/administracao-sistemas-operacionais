  1. Responda certo ou errado e justifique: “A arquitetura de software tem como objetivo apresentar a estrutura dos componentes lógicos do sistema, ficando a análise de dados para a disciplina de AD (Active Directory)”

    2. Errado. A arquitetura de software abrange a estrutura geral do sistema, incluindo seus componentes lógicos, como eles interagem e como os dados são armazenados e acessados. A análise de dados é uma parte importante da arquitetura, definindo modelos de dados e estratégias de persistência. O Active Directory (AD) é um serviço específico para gerenciamento de identidades e acesso em redes Windows, não a disciplina responsável pela análise geral de dados do sistema.

  2. O que é HYPER e qual a diferença para VirtualBox?

    3. HYPER provavelmente se refere ao Hyper-V, a plataforma de virtualização nativa da Microsoft, integrada ao Windows Server e algumas edições do Windows. A principal diferença para o VirtualBox, que é um software de virtualização de terceiros (Oracle), é que o Hyper-V é um hypervisor de tipo 1 (bare-metal) em sistemas server, rodando diretamente sobre o hardware (embora em sistemas desktop funcione como tipo 2). VirtualBox é geralmente um hypervisor de tipo 2 (hosted), rodando como uma aplicação sobre um sistema operacional já instalado. Hyper-V tende a ter melhor integração com o ecossistema Microsoft e pode oferecer melhor performance em certos cenários Windows.

  3. O que é uma floresta dentro do AD/ADDS?

    4. Uma floresta no Active Directory Domain Services (AD DS) é o contêiner de nível mais alto na estrutura lógica do AD DS. Ela representa uma ou mais árvores de domínio que compartilham uma relação de confiança transitiva.

    Pense em uma floresta como uma coleção de domínios que confiam uns nos outros. Essa confiança permite que os usuários de um domínio acessem recursos em outro domínio dentro da mesma floresta, dependendo das permissões configuradas.

  4. Faça um diagrama/desenho de como se organizam Unidade Organizacionais: TURMA e Usuários: DOIS COLEGAS [árvore]

    5. Uma floresta dentro do AD/ADDS é uma coleção de um ou mais domínios do Active Directory que possuem relações de confiança transitivas entre si. Ela representa uma estrutura organizacional maior, permitindo o compartilhamento de recursos e informações entre os domínios da floresta, mantendo, ao mesmo tempo, limites administrativos separados.
